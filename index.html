Sobre os Algoritmos

Conheça os diferentes métodos de criptografia disponíveis

Base64

Codificação que converte dados binários em texto ASCII. Amplamente usado para transmissão de dados na web.

Caesar Cipher

Cifra de substituição onde cada letra é deslocada por um número fixo de posições no alfabeto.

ROT13

Variação do Caesar Cipher com deslocamento fixo de 13 posições. Aplicar duas vezes retorna o texto original.

Reverso

Método simples que inverte a ordem dos caracteres do texto. Fácil de implementar e entender.
